Ecommerce 'The Scent' Store

Technologies: HTML, CSS, JavaScript, MySQL, PHP

Leveraging a real-life e-commerce scenario, I developed a website to manage perfumes using a MySQL database, maintaining real-time inventory for customer interactions. Items are effortlessly added to the shopping cart with an automatic total sum calculation.

The store enhances user experience and backend efficiency by using foreign keys to link the product information table with the stocks and carts tables. This setup allows administrative staff to quickly access perfume details by clicking on the perfume ID in the stocks table, optimizing inventory management. Additionally, it offers up-to-date information on stock availability, ensuring accurate and timely details for customer service.

A perfume e-commerce website with a MySQL database managing real-time inventory. Items are easily added to the cart with automatic total calculation.

Product information links to stock and cart tables via foreign keys, allowing admin staff to access perfume details with a single click. This setup enhances inventory management and improves customer service by ensuring that customers cannot purchase items that are out of stock.

  • Developed an e-commerce website for managing perfume inventory using MySQL.
  • Implemented real-time stock checks and automatic shopping cart calculations.
